# 文音悦读

[每日一文](https://meiriyiwen.com/) 非官方 Android APP

采用 Kotlin 语言编写，配合 MVP 模式探索。

用到的第三方框架主要有：

- RxJava
- RxAndroid
- RxBinding 系列
- RxLifecycle
- Dagger 2
- AutoValue 系列
- Jsoup
- ExoPlayer
- SQLDelight
- Retrofit 2
- OkHttp 3
- Picasso
